DESCRIPCIÓ DEL PROJECTE: En aquest projecte hem creat una pagina web utilitzant html i css, dintre de la nostre pagina web expliquem i donem un resum directe del CRM de SAP (SAP CX), la pagina web es composa de 4 seccions; primer el inici, en aquesta secció introduim els punts clau destaquen de SAP CX.
Despres la secció de features es donen una descripcio de les eines que utilitza SAP CX, acompanyat d'una taula la cual mostre un comparació d'integritat, personalitzció, escalabilitat, i preu anual amb eltres crm destacats.
Despres tenim la secció de Pricing on mostrem els preus i una descripcio informativa de cada eina, i per ultim tenim la secció de suport destinada a usuaris que puguin experimentar alguna anomalia amb SAP CX, amb la secció de suport podren consultar i demanar ajuda su o necessiten.
